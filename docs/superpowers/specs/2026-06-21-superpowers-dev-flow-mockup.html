Superpowers 开发链 → Synod Flow

头脑风暴 → spec → 计划 → subagent 开发 → review，编码成「按产物切的子 flow 链」。强模型=codex，开发=deepseek。父 flow superpowers.mjs 用 runWorkflow 串四段。

⑂ 父 flow · superpowers.mjs — runWorkflow 串 4 段 + 接缝人审(可配)

1 头脑风暴 → spec codex 对话

workflows/brainstorm-spec.mjs
子 agent 自适应提问，人作答，循环到「两把钥匙」都到位才结束。JS 不懂语义，只查记号 / 人决定 / 轮数。
↻ 提问循环（人在环豁免硬上限；MAX 仅防失控）
codex 问下一个澄清问题 reuse 会话 → 🟢 人作答 ask() → 喂回 transcript ↩︎ 再问
判定「头脑风暴结束了吗」= JS 只查两把钥匙 + 刹车
① codex 吐 <<<SPEC>>> 记号
提议：够了 + 草稿 → ② 🟢 人 accept
拍板：才真结束 → ✅ 写 spec.md
↩︎ 人不 accept、给反馈 → 草稿没过，带反馈接着聊
⛔
刹车：到 MAX 轮 / 人打 /spec → 强制收尾，别让 codex 一直问下去
钥匙① = agent 提议（记号）钥匙② = 人拍板（accept）刹车 = MAX / /spec
⚠︎ 记号只是提议，不盲信（codex 未必每次老实吐）；人 accept 才算数。提问取人答要新原语 ask()——只返回人打的原始整行，不像 approve 把「ok / 空行」误判成 accept / abort。这是要写进 spec 的新增项。
▼ 产物 · spec.md
人审卡口 · 仅 gates = all

2 写计划 codex

workflows/spec-to-plan.mjs
读 spec 产出 TDD 计划（task 列表），人在环改稿定稿。
📝
agent(codex) 由 spec 产出分 task 的实现计划
🟢
reviseWithHuman() 人给自然语言反馈→改→定稿 人
💾
→ plan.md
▼ 产物 · plan.md (task 列表)
人审卡口 · 仅 gates = all

3 subagent 驱动开发 deepseek 写codex 审最肥

workflows/execute-plan.mjs
parsePlan() 解析 task 列表 → JS for 逐 task；并行写用 workspace git worktree 隔离。
↻ 每个 task：backtrack 循环（≤ 3 轮）
deepseek 写码 write:true · worktree → bash npm test → codex 审 write:false → PASS?
↩︎ 不过：把「测试输出 + codex 反馈」喂回 deepseek 重写（带反馈回退，非回滚世界）
⛔
自动刹车：3 轮仍红 / codex 仍拒 → 停在此 task，不往下合（绝不硬塞坏码）
🟢
每 task 验收 approve() 人 · 仅 gates = all
▼ 产物 · diff（worktree：干净自动合 / 冲突留存待你）
人审卡口 · 仅 gates = all

4 最终 code review codex

workflows/final-review.mjs
对全量 diff 做一遍独立评审，不过则带反馈修。
🔎
agent(codex, write:false) 审全 diff
🔧
backtrack 有问题 → deepseek 修 → 复审
✅ 最终人审 · gates ≥ final（默认建议留这一处）
合并 / 开 PR

角色 / 图例

codex — 强模型：brainstorm · plan · review
deepseek — 开发写码（write:true）
人 — 头脑风暴对话 + 可配人审卡口
bash / npm test — 自动验证
产物 — 子 flow 间的纯数据交接（= 天然边界）

人审卡口 = input 开关

none全自动零人审 — 你的选择（默认）
final只最终合并前停一次（几乎零成本，建议留）
allspec后 / plan后 / 每task后 / 最终（superpowers 原意）

绿色虚线卡口随档位开关；codex 评审 + npm test 永远在，那不是你、跳不掉。

你走开后发生什么

跟它头脑风暴聊出 spec → 然后走开 → 它自己 plan、deepseek 写码、codex 审、跑测、合并 → 你回来收一个干好的 PR。过不了测试/评审就停下等你，不闭眼合。
